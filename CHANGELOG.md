## [0.1.0] - 2024-01-XX

### Added
- Initial release of kubectl-rltop
- Display pod CPU and memory usage from Metrics API
- Display CPU and memory requests and limits from pod specifications
- Support for namespace filtering (`-n`, `--namespace`)
- Support for label selector filtering (`-l`, `--selector`)
- Version command (`kubectl-rltop version`)
- Comprehensive documentation
- CI/CD pipeline with GitHub Actions
- Krew plugin manifest
